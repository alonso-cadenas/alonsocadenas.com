- [Sakura](https://github.com/oxalorg/sakura/)

## How To Use

### Installation

Install package dependencies

```bash
npm install
```

### Local Development

Run in development mode at http://localhost:5173/.

```bash
npm run dev
```

### Build

Build for production.

```bash
npm run build
```

### Preview

Locally preview the production build prior to deployment.

```bash
npm run preview
```
