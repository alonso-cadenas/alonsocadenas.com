## Technologies Used

- [Vite](https://vite.dev/)
- [Prettier](https://prettier.io/)

## How to use

### Installation

Install package dependencies

```bash
npm install
```

### Local Development

Run in development mode at http://localhost:5173/.

```bash
npm run dev
```

### Build

Build for production.

```bash
npm run build
```

### Build

Locally preview the production build prior to deployment.

```bash
npm run preview
```
